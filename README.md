# Weather forecast

This app uses OpenWeather API to show the current weather for the selected city and a 5-day forecast.


## How to run:
### 1. Install Node.js
   Inside the main folder install node modules using npm.
### 2. API key
   Get your unique API key here: https://openweathermap.org/api 
### 3. Create .env file
   Create a new file and save it as .env <br>
   Inside this file, add this line: `API_KEY=your_API_key`
### 4. Run server.js
   To run server.js, simply type: `node server.js` into the terminal.
### 5. View in browser
   Type `localhost:3000` in your browser to view the application.
